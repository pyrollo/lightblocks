Light Blocks
------------

Light Blocks provides several light source decorative blocks.

If pipeworks or homedecor mod is present, Light Blocks will use homedecor:plastic_sheeting. Else it will define homedecor:oil_extract, homedecor:paraffin and homedecor:plastic_sheeting. 

Code and textures for oil, paraffin and plastic have been taken from VanessaE pipeworks mod.
